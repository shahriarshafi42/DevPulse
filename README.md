#  DevPulse – Internal Tech Issue & Feature Tracker
Live URL: devpulse-one-rho.vercel.app


# Project Overview
DevPulse is a collaborative issue tracking system designed for software teams to report bugs, request features, and manage workflows efficiently. It supports role-based access control, JWT authentication, and PostgreSQL-based data storage.


# Features
- User registration and login with JWT authentication
- Role-based access control (Contributor & Maintainer)
- Create, view, update, and delete issues
- Secure password hashing using bcrypt
- Protected API routes with middleware authentication
- Clean modular Express.js architecture
- PostgreSQL database with raw SQL queries


#Tech Stack
- Node.js (LTS 24+)
- Express.js
- TypeScript
- PostgreSQL
- bcrypt 
- jsonwebtoken
